## RESUMO PARA MEMORIZAÇÃO RÁPIDA

### Teto
- Fórnice → Tela superior → Velum → Tela inferior → Plexo
- VCIs + Coroidais posteriores mediais no velum

### Assoalho
- Quiasma → Infundíbulo → Túber → Mamilares → Mesencéfalo
- 4 recessos: óptico, infundibular, suprapineal, pineal

### Paredes Laterais
- Sulco hipotalâmico divide: Tálamo (cima) / Hipotálamo (baixo)
- Massa intermédia presente em ~70%

### Parede Anterior
- Colunas do fórnice + Comissura anterior + Lâmina terminal

### Parede Posterior
- Pineal + Comissura posterior + Aqueduto

### Forame de Monro
- Anterior/Superior: Coluna do fórnice
- Posterior: Tálamo
- Talamoestriada passa pela margem posterior

### Abordagens
- Transcaloso-interforniceal: pelo meio do fórnice
- Transchoroidal: pelo lado fornical da fissura
- Translaminar terminal: pela lâmina terminal (subfrontal)
